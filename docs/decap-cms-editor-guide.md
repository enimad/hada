# Guide simple pour publier un article avec Decap CMS

## Avant de commencer

Il faut :

- un compte GitHub gratuit ;
- une invitation acceptée sur le repo GitHub `enimad/hada` ;
- l'adresse d'édition : `https://hadawedding.fr/admin`.

## Créer un article

1. Ouvrir `https://hadawedding.fr/admin`.
2. Cliquer sur la connexion GitHub.
3. Se connecter avec le compte GitHub invité sur le repo `enimad/hada`.
4. Ouvrir la collection `Articles du Blog Hada`.
5. Cliquer sur `New Article`.
6. Remplir les champs : titre, slug, description, catégorie, date de publication, image principale, SEO et contenu.
7. Enregistrer puis publier.

## Programmer un article

Pour programmer un article, il suffit de choisir une date future dans `Date de publication`.

L'article sera enregistré dans le site, mais il ne sera visible publiquement qu'à partir de cette date.

## Images et vidéos

Les images peuvent être ajoutées directement dans Decap. Elles seront rangées dans le dossier média du projet Hada.

Les vidéos lourdes ne doivent pas être envoyées dans Decap, car elles alourdissent le repo GitHub et le site. Il faut les mettre sur YouTube, Vimeo, Loom ou Google Drive, puis coller le lien dans le champ `Lien vidéo optionnel` ou dans le contenu de l'article.

En résumé : image dans Decap, vidéo via lien externe.

## À éviter

- Ne pas changer le slug d'un article déjà publié sauf si une redirection est prévue.
- Ne pas uploader de vidéos lourdes dans le dossier média.
- Ne pas publier sans image principale ni description SEO.
